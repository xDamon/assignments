Directories and Files:

samples - Folder of sample input files for the turtle graphics program
src - The source code for the turtle graphics program, as well as the Makefile to build it
DeclarationOfOriginality - Picture of my signed declaration of originality
Report - PDF containing the assignment report

Compiling:

The makefile has targets for each program name, TurtleGraphics, TurtleGraphicsSimple and TurtleGraphics debug.
It also has shorter named rules like just make, make simple, make debug for each option respectively.
